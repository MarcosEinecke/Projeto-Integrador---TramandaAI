# TramandaAí

## Projeto Integrador - Desenvolvimento em Sistemas

Bem-vindo ao repositório do projeto TramandaAí, uma aplicação web desenvolvida como parte do projeto de conclusão do curso de Tecnologia da Informação no Senac. O objetivo principal é transformar a experiência turística em Tramandaí por meio de um guia digital moderno, acessível e informativo.

---

## 🌊 Sobre o projeto

TramandaAí é um projeto de autoria dos alunos Luís Henrique Maicá de Souza e Marcos Einecke, criado para auxiliar turistas e a própria comunidade local. A proposta é oferecer um guia turístico digital focado na divulgação de pontos importantes da cidade, permitindo que os usuários conheçam melhor os principais espaços, serviços e atrações de forma prática e interativa.

Ao escanear QR codes espalhados por Tramandaí, os usuários terão acesso a uma página com:

- galeria de fotos;
- descrição do local;
- endereço;
- horário de funcionamento;
- mapa interativo.

Essa solução busca resolver a dificuldade de comunicação frequente em cidades de médio e pequeno porte, onde as informações turísticas costumam estar desatualizadas em panfletos impressos ou em sites antigos.

---

## ✨ Funcionalidades principais

O sistema contará com as seguintes funcionalidades:

- Página inicial com artigos e destaques sobre os pontos turísticos;
- Página de detalhes do ponto turístico, contendo descrição, endereço, horário de funcionamento, galeria de fotos e mapa;
- Painel administrativo com CRUD completo para gerenciamento dos pontos turísticos;
- Página institucional “Sobre a cidade”;
- Página de feedback;
- Layout totalmente responsivo, pensado para dispositivos móveis.

---

## 🚫 O que o sistema não contempla

Nosso sistema dispensa:

- aplicativo mobile nativo, pois o acesso será realizado por navegador;
- sistemas de reserva, venda de ingressos ou qualquer tipo de interação com meios de pagamento;
- suporte a múltiplos idiomas, estando disponível apenas em PT-BR.

---

## 🛠️ Arquitetura prevista

A estrutura do projeto será composta por:

1. Frontend: HTML5, Tailwind CSS e JavaScript;
2. Backend: PHP com padrão MVC e conexão via PDO;
3. Banco de dados: MySQL;
4. Ferramentas de apoio: GitHub, Trello e Canva.
